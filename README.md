<h2>Описание функционала приложения</h2>
Основная решаемая задача - расчёт равномерности нанесения
покрытия из косинусного источника (согласно закону Ламберта-Кнудсена).
<br>
<br>
Источник может быть:
<ol>
<li>Точечный</li>
<li>В виде множества точек вдоль окружности</li>
<li>В виде множества точек, распределённых по алгоритму подсолнуха
внутри области, ограниченной окружностями заданных радиусов</li>
<li>С помощью цикла можно задать распределение источников 
вдоль интересующей нас траектории. В том числе, заданной в пространстве</li>
</ol>

Подложка может быть плоской замкнутой фигурой (задание координат фигуры и положения плоскости
не ограничено).
<br>
<br>
Подложка может осуществлять вращение с постоянной угловой скоростью относительно заданной оси.
<h3>Применение маски для корректировки равномерности</h3>
Если часть потока срезать маской, то равномерность увеличится.
Программа включает возможность вставки препятствия для потока материала (лучей).
Препятствием (маской) может быть плоская замкнутая фигура. <br><br>Выбор плоскости и замкнутой фигуры не ограничен.
<br><br>Получение *реальной* формы маски возможно благодаря её пропорциональному проецированию на плоскость,
высота которой определяется конструктивно.
